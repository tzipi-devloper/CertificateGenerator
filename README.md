# Certificate Generator

מחולל תעודות אוטומטי המייצר קבצי PDF מותאמים אישית לעובדים על בסיס נתוני CSV.

## תכונות

- עיבוד נתוני עובדים מקובץ CSV
- יצירת תעודות PDF מותאמות אישית
- מערכת לוגים מקיפה
- טיפול חכם בשגיאות
- תמיכה בתבניות Word

## קבצים נדרשים

- `Data.csv` - נתוני העובדים
- `Template.docx` - תבנית התעודה
- Microsoft Word (עבור COM Interop)

## הרצה

1. הורד את הפרויקט
2. פתח ב-Visual Studio
3. וודא שקבצי Data.csv ו-Template.docx נמצאים בתיקיית הפרויקט
4. הרץ את הפרויקט

## פלט

התעודות נוצרות בתיקיית `Output` עם שמות הקבצים: `FirstName_LastName.pdf`

## לוגים

הפעילות נרשמת בקובץ `application.log`
